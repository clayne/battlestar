Assign a value to a register:

    register = value

    example:
	eax = 2

Declare a function:

    fun name

    example:
	fun hello_world

    example:
	fun hello
	    eax = 2
	    ret(0)

Return from a function:

    ret [retval]

    example:
	ret(1)

    example:
	ret

Declare a constant:

   const name = constant
