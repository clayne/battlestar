Assign a value to a register:

    register = value

    example:
	eax = 2

Declare a function:

    fun name

    example:
	fun hello_world

    example:
	fun hello
	    eax = 2
	    ret(0)

Return from a function:

    ret [retval]

    example:
	ret(1)

    example:
	ret

Declare a constant:

   const name = constant 

   The given value can be a string, comma-separated list of values or a mix of both

Built in functions:

    len(name)

    Represents the length of a constant, by the given name
